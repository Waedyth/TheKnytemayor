Command Basics/ give

The give command has 3 paramaters - a target selector (the player to give the item to), itemstack (the item's ID and components), and an int (the item count). Only the first two paramaters are required. If the last paramater is ignored, the amount of items will be 1.


Here is an example:


/give @p minecraft:apple[] 1

Of course, that is a very basic example. You can modify the item's properties using the components in the itemstack paramater.


/give @p minecraft:apple[ item_name= "Cripps Pink Apple", enchantment_glint_override= true ] 12
